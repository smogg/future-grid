Future Grid

Responsive, fluid, mobile-first grid system based on flexbox and built using SASS. Contains predefined Bootstrap-inspired classes & SASS mixins.

By default, grid is made of 12 columns and takes 100% of the available width. Columns should be placed inside container with a .grid class. One .grid container can contain any number of columns.

Holy Grail Layout

Two fixed width sidebars on the sides and fluid width content. Content should go first in the markup but should be visible as the middle one. All three columns should be of equal height. Everything should fall down to signle-column layout for smaller screens.

Read more: Solved by Flexbox: Holy Grail Layout (Philip Walton)

content
left sidebar
right sidebar

<div class="grid">
  <div class="col-12 col-sm-auto">
    <div class="content-center content">content</div>
  </div>
  <div class="col flex sidebar col-order-first">
    <div class="content-center">left sidebar</div>
  </div>
  <div class="col flex sidebar">
    <div class="content-center">right sidebar</div>
  </div>
</div>
            

.content {
  height: 220px;
}

.sidebar {
  width: 100%;
}

@media(min-width: 500px) {
  .sidebar {
    width: 150px;
  }
}

Overflow Pattern

Make a grid become a scrollable list of items on smaller screens by adding a class of .slider‑below‑lg. Don't forget, to provide hints informing user that there's more content available.

Read more: The Overflow Pattern (Brad Frost)

SASS Variables & Mixins

You can use SASS mixins to build your own semantic layouts. Refer to the grid project folder and read the source code to find out more.

Browser Support

Future Grid uses Bourbon.io mixins. They are up to date with 2014 W3C Spec and include fallbacks for 2009 spec.

Future grid should work in all modern browsers and IE10+.
